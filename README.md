# docker-imapsync

## Usage

```bash
docker run choffmeister/imapsync:latest \
  --host1 server1 \
  --user1 user@server1 \
  --password1 user1password \
  --ssl1 \
  --host2 server2 \
  --user2 user@server2 \
  --password2 user2password \
  --ssl2
```
